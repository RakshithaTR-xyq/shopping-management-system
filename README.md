# 🛒 Shopping Management System

A Shopping Management System developed using **Node.js**, **Express.js**, and **MySQL** for managing shopping-related database operations.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## 📌 Project Overview

This project is a backend application that demonstrates how to perform CRUD (Create, Read, Update, Delete) operations using Node.js, Express.js, and MySQL. It connects to a MySQL database and provides functionalities for managing shopping-related data.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## 🚀 Features

- Add new shopping records
- View shopping records
- Update existing records
- Delete shopping records
- MySQL database connectivity
- REST API implementation using Express.js

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## 🛠️ Technologies Used

- Node.js
- Express.js
- MySQL
- JavaScript

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/RakshithaTR-xyq/shopping-management-system.git
```--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### Navigate to the project folder

```bash
cd shopping-management-system
```--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### Install dependencies

```bash
npm install
```

### Start the server

```bash
node index.js
```
